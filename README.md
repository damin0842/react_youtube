# Getting Started with Create React App   

## react start   

npm create-react-app ./   

## axios 설치   

npm i axios   

## router 설치   

npm i react-router-dom    

## react player 설치    

npm i react-player   

## scss 설치   

npm i node-sass   
npm i sass   

## react icon 설치   

https://react-icons.github.io/react-icons/   
npm install @react-icons/all-files --save
